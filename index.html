Get BibTeX from a DOI — fast.

QuickBib is a small cross-platform app that uses doi2bib3 to fetch accurate BibTeX for DOIs and arXiv identifiers. Works as a GUI app and also as a CLI. Perfect for research workflows, manuscripts and automation.

Install on GNU/Linux Download for Windows/MacOS

Why QuickBib?

Designed for researchers, writers and automation — QuickBib makes fetching and normalizing BibTeX simple.

Quickly obtain BibTeX

Uses doi2bib3 strategies (content negotiation, Crossref, arXiv resolution) to return high-quality BibTeX entries.

arXiv-aware

Paste arXiv IDs, arXiv URLs, or legacy identifiers — QuickBib detects them and resolves the DOI automatically.

GUI + CLI

Use the GUI app for quick lookups or integrate into scripts via the CLI and programmatic API.

Packaged for GNU/Linux, Windows and MacOS

Available via AUR on Arch and a PPA for Ubuntu. Also available as prebuilt installers on Releases for Windows and macOS.

How to install

Choose the option that fits your platform.

GNU/Linux

Ubuntu (Official PPA)

sudo add-apt-repository ppa:apandada1/quickbib
sudo apt update
sudo apt install quickbib

Arch Linux (AUR)

yay -S quickbib

From source (any distro)

QuickBib depends on doi2bib3 for DOI resolution and usual Python/GUI dependencies. Build and install with meson:

git clone https://github.com/archisman-panigrahi/quickbib.git
cd quickbib
meson setup builddir --prefix="$HOME/.local"
meson install -C builddir

# to uninstall
meson uninstall -C builddir

If you need doi2bib3, see its README (the backend provides the CLI doi2bib3 and a Python API).

Quick examples

QuickBib supports all of the input types shown below (DOIs, DOI URLs, arXiv IDs/URLs, Journal URLs). The CLI (doi2bib3) can also write output directly to BibTeX files using -o / --out.

CLI: doi2bib3

doi2bib3 10.1038/nphys1170
doi2bib3 http://dx.doi.org/10.1038/nphys1170
doi2bib3 https://arxiv.org/abs/2411.08091
doi2bib3 arxiv:2411.08091
doi2bib3 2411.08091
doi2bib3 https://journals.aps.org/prl/abstract/10.1103/v6r7-4ph9
doi2bib3 2411.08091 -o paper.bib

Programmatic (python)

from doi2bib3 import fetch_bibtex
bib = fetch_bibtex('https://doi.org/10.1073/pnas.2305943120')
print(bib)

For details on the programmatic API and options, see the doi2bib3 README.
